# About

    Program name: „Find usage”
    Programming language: Bash
    Author: Łukasz Gołojuch
    Operating system: Linux Knoppix
    
## Program description

The program mainly uses find commands to find folders with given params.

## Usage

findLogs() - function that finds all logs under certain direction

findEmptyFiles() - function that finds all empty files under certain direction

findEditedFiles() - function that find edited files in give time frame and under certain direction

findHiddenFiles() - function that finds all hidden files under certain direction

findExecutable() - function that finds all exec files under certain direction

runProgramWithout777() - function that run bash program that do not have right permission granted. It gives 777 permission and than run this program.
